# 🌟 Overview

**CareerPilot AI** is an AI-powered career development platform that helps students and job seekers prepare for successful careers through personalized guidance, resume optimization, interview preparation, job discovery, and skill development.

The platform combines intelligent recommendations with an intuitive user experience to support every stage of a user's career journey—from learning new skills to landing their dream job.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/yourusername/CareerPilot-AI.git
```

Navigate

```bash
cd CareerPilot-AI
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```
