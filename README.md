# 🚀 CareerConnect – AI Powered Career Development App

CareerConnect is a modern Android application designed to help students and job seekers accelerate their careers. The app brings together job discovery, resume building, interview preparation, skill development, and career tracking in one intelligent platform.

---

## ✨ Features

### 💼 Job Discovery

* Search jobs and internships
* Filter by location, role, salary, and experience
* Save favorite jobs
* Apply directly from the app

### 🤖 AI Resume Builder

* ATS-friendly resume generation
* Multiple professional templates
* AI-powered resume suggestions
* Resume score analysis
* PDF export

### 🎤 AI Interview Preparation

* HR interview questions
* Technical interview practice
* Mock interviews
* AI feedback
* Confidence analysis

### 📚 Skill Development

* Learning roadmap
* Course recommendations
* Progress tracking
* Certifications management

### 👨‍💻 Career Dashboard

* Job application tracker
* Interview schedule
* Saved companies
* Career progress analytics

### 🌐 Community

* Share achievements
* Connect with peers
* Career discussions
* Networking opportunities

### 🔔 Smart Notifications

* New job alerts
* Interview reminders
* Application updates
* Skill recommendations

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/CareerConnect.git
```

2. Open the project in Android Studio.

3. Sync Gradle dependencies.

4. Add your Firebase configuration (`google-services.json`).

5. Run the application on an emulator or Android device.
